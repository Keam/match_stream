Overview

Match Streamer lets you watch your favourite soccer matches in your terminal :)

Usage

  • To start it run $ make run
  • To connect to the server run $ telnet 9999
  • Once connected, to watch a game send VERSION:1:CONNECT:your-name:MATCH:the-match
  • To know the available matches open http://localhost:8888/1/matches on your browser
  • You can also get detailed information about a match using http://localhost:8888/1/match/match-id
  • You can see its history using http://localhost:8888/1/match/match-id/history
